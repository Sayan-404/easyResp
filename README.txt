A simple python script to automate recon and gather http/https status codes and
collect headers and put everything in a file neatly no super knowledge of vim
is required.

Only *.txt files are supported for file inputs and outputs

How to use:

 -h, --help             help
  -u URL, --url URL     Returns the status code
  -f FILE, --file FILE  Takes a list of input
  -o OUTPUT, --output OUTPUT
                        gives output in a text file please mention .txt at the
                        end
  --header HEADER       include header type y for yes
